üí° Example Requests

Create Car 
curl -X POST http://localhost:3000/api/cars \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer secret123" \
  -d '{
    "brand": "Toyota",

    "category": "sedan",
    "plate": "XYZ-789",

  }'

  -H "Content-Type: application/json" \
  -d '{

    "customer": {
      "name": "John Doe",
      "email": "john@example.com"
    },
    "from": "2025-11-01",
    "to": "2025-11-05"
  }'
